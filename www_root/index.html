Congrats, your webserver is healthy. If you're the app owner, configure the www root to serve your webpage.

If you're running the app in Docker and want to serve files from a local directory on your machine, say on '/tmp/webserver_public', use the -v flag to define a volume and the -e flag to configure the www root directory.

docker run -p 8080:8080 -v ~/tmp/webserver_public:/local_www_root_dir -e www.root.dir=/local_www_root_dir lemberski/webserver
